正则表达式

参考博客python正则表达式
Python中在匹配模式字符前加上’r’表示 “raw” string，其会在不改变反斜杠（即不用对其转义）的情况下通过（特别是在linux和windows系统的路径不同的时候特别有用），因此建议在写正则表达式时在前面加上’r’作为一个习惯。

概念
